Format notes

CSV
  • Headers: ID, Website/App Name, Investment Type, Sub Type Name, Sub Type Category, Amount, Investment Date
  • Date format: YYYY-MM-DD
  • Matching records (platform + type + sub-type + category) are updated
SQL
  • Exports all tables: investments, investment_history, sub_type_names, sub_type_categories, investment_transactions
  • Merge: adds data without clearing existing records
  • Fresh install: truncates all tables then imports — like a clean setup
MongoDB
  • Extended JSON format with all portfolio collections
  • Compatible with mongoimport and MongoDB tools
  • Works when backend uses MySQL or MongoDB (DB_TYPE)
  • Merge: upserts by document ID
  • Fresh install: clears all collections before import
